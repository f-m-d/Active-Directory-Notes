"Over pass the hash" genera token dagli hash.

L'attacco classico "Pass the hash" permette di usare l'hash NTLM
per permetterci di autenticarci a server/macchine che supportano
l'autenticazione tramite hash NTLM della password.
Anche le macchine locali accettano hash di tipo NTLM.

"Over pass the hash" invece crea dei Ticket Kerberos validi,
a partire dall'hash NTLM.
